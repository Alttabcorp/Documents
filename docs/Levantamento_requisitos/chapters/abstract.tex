\begin{abstract}
Este documento apresenta a especificacao completa do projeto \textbf{FinOps}, uma aplicacao web e mobile voltada para organizacao financeira pessoal e familiar. O sistema tem como objetivo principal facilitar o controle de receitas, despesas fixas e variaveis, poupanca, reserva de emergencia, investimentos e gastos livres, proporcionando uma visao clara e organizada das financas para ate duas pessoas, com divisao equitativa dos recursos disponiveis.

A aplicacao visa resolver os desafios comuns de gestao financeira pessoal, oferecendo uma interface intuitiva e ferramentas praticas para planejamento orcamentario, acompanhamento de metas financeiras e tomada de decisoes baseadas em dados reais. O projeto contempla tanto o desenvolvimento de uma versao web quanto mobile, garantindo acessibilidade e praticidade para os usuarios em qualquer momento e lugar.
\end{abstract}
